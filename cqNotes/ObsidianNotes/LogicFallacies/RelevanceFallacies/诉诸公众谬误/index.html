诉诸公众谬误

Created2022-05-07|Updated2022-06-16|CriticalThinking
|Reading time:9min

庞葱与太子质于邯郸，谓魏王曰：“今一人言市有虎，王信之乎？” 王曰：“否。”“二人言市有虎，王信之乎？” 王曰：“寡人疑之矣。”“三人言市有虎，王信之乎？” 王曰：“寡人信之矣。” 庞葱曰：“夫市之无虎明矣，然而三人言而成虎。今邯郸去大梁也远于市，而议臣者过于三人矣。愿王察之矣。” 王曰：“寡人自为知。” 于是辞行，而谗言先至。后太子罢质，果不得见。

–《战国策・魏策二》

吾尝闻大勇于夫子矣。自反而不缩，虽褐宽博，吾不惴焉；自反而缩，虽千万人，吾往矣。

– 《孟子・公孙丑上》

诉诸公众谬误指论证者利用一个人在群体中的某种心理和情绪（比如盲从，虚荣，势力等）为基础，让人接受某个结论，而不是用可靠的证据以及恰当的逻辑推理。

谈起诉诸公众谬误，最先涌现我脑海里的是阿希从众心理学实验。这个实验是由著名社会心理学家所罗门 ・阿希设计的一个非常经典、有影响力的心理学实验。

实验问题很简单，皆为常识性且不需要专业知识，比如：“左图中的线条在长度上最接近右图中的哪一条？”

在这个简单的实验中，共有 124 位不同高校的男性学生参与。这些学生以小组（每组 6-9 人）为单位参与答题，其中每组中仅有一人是真正参与实验的志愿者，其余皆为实验研究员。研究员都故意回答错误（比如一致回答错误答案 A 而不是正确答案 C），真正的志愿者则在临近最后回答问题。当志愿者给出与他人不同的答案时往往会引来其他人的注视。

面对这样简单的问题，约三分之一的参与者一直附和他人，给出错误的答案；而 75% 的人在 12 轮的实验中至少给出了 1 次错误答案，仅有 25% 的志愿者坚持了自己的判断。相比之下，如果参与者独立判断，答题的错误率不到 1%。

令人感到不安的是，参与者都是来自高等院校的学生，拥有良好的教育背景，而给出与众不同的答案时引来的也仅仅是别人的注视。试想如果是在一个复杂的社会，教育程度参差不齐，引来的也不仅仅是注视，那结果恐怕更加触目惊心（让人不禁联想到皇帝新装故事里那些吹嘘拍马称赞皇帝盛装的臣民们，大跃进时期踊跃跟进虚报粮食产量的村庄，二战德国大呼 “希特勒万岁” 的党卫队，以及文革期间强横暴力甚至残害他人却鲜有愧疚感的红卫兵们， 等等）。

一个人的情绪和非理性在一个群体中更容易被调动起来。现实生活中，往往每个人都有得到他人的认同、尊重、赞赏以及害怕被排斥、出丑、嘲笑的心理。这种情绪和心理便成为了诉诸公众谬误的基础。在缺乏理性和逻辑常识的情况下，具有煽动性且不断重复的措辞往往会让很多人情绪高涨、潜移默化地接受一些观点，而且效果好得惊人，虽然这些口号或观点背后可能并没有充分的理由和严密的逻辑论证。

与此同时， 大众观点对个人的判断又会产生影响，如阿希从众心理学实验所揭示的那样。开篇引言关于 “三人成虎” 的成语故事也描述了类似的情况: 当多数人说街市上有虎，虽然明知街市上没有老虎， 但魏王还是从 “寡人疑之” 变成了 “寡人信之”。

总而言之，诉诸公众谬误的基本结构是：一部分人 (或大多数) 赞同某个观点，或在做某件事；所以，这个观点或行为就是正确的 (因此，你也应该接受)。

这和之前介绍过的一厢情愿谬误相似。 如下图，一厢情愿谬误的结构为: 我认为、相信、或希望某件事或某个观点 (P), 所以这件事或观点正确 (左); 诉诸公众谬误则把个人换成来一部分人或某个群体 (右)。

当然，人数多少与某个结论的正确与否并没有必然逻辑关系，大多数人都接受某个流行的结论并不能充分证明这个结论正确合理。不幸的是，惯性的诉诸公众谬论往往伴随着一种危险且扭曲的心态: 因为大家都在做，所以我也做，即便错了我也不用承担任何后果、受到道德谴责。 如伏尔泰评价的那样：“No snowflakes in an avalanch ever feels responsible （雪崩中的每一片雪花都不认为自己负有任何责任）”，即使这场雪崩夺去无辜的生命，造成不可挽回的悲剧。

这种对个人心理和情绪的影响及操纵可以是直接的，也可以是间接的。直接的诉诸公众谬误经常以口号标语的形式出现 (比如” 希特勒万岁 “，” 打倒走资派刘少奇 “, “打倒美帝国主义”，“两个凡是”，等等)。间接的诉诸公众谬误论证方式利用的是群体、或个人与群体的某种关系，常见的有以下几种。

诉诸时尚：如果你不接受 X，那么你就会落伍或赶不上某种潮流。

X 牌洗面奶是你毫无疑问的选择，因为 95％的人都在用它。

上例中，选择 X 牌洗面奶的理由是大多数人（95%）都在用。如上所述，人数的多少并不能充分证明这个选择的合理性。且不论上述数据是否真实，仅仅大多数人用 X 产品的前提也不能充分推出 “你也应该选择这款产品”。与上例相关的论据应该是这种产品质量是否合格，是否满足自己的各种要求，是否价格合理，等等。 （参看遮盖论据谬误）

诉诸虚荣，势力：如果接受 XYZ，你将拥有某种令人羡慕的品质，受到追捧，或融入某个 “高人一等” 的圈子。

X 牌早餐奶是你不二的选择。喝 X 牌早餐奶，拥有刘翔的速度，爱因斯坦的大脑，以及姚明的身高。

这里前提是如果你选择 X 牌早餐奶，你会拥有一些令人羡慕的特质，比如健康的体魄或聪明的头脑。然而，前提除了利用大众对得到这些特质的希望，并没有列出更为相关的直接证据，所以前提不能充分支持起结论。

选择这块 X 系列手表绝对值。不是每个人都有资格佩戴它：它是身份的象征，一般只有千万身价以上的富人才带 X 系列手表。所以，为了成为少数有资格佩戴这块手表的人之一，你应该买下我这块。

融入一些在某方面 “高人一等” 的圈子（比如富商大贾、或其它有权势的组织）在很大程度上是一种普遍心理。如果评判论证标准是这个圈子固有的理念或规定，和加入这个圈子的势力心，而不是理性逻辑，那么这种论证就属于诉诸势力谬误。

诉诸传统：因为某观念（或行为）持续了很久，已成为传统或习惯；因此，这种观念（以及其支持的相关结论）是好的、正确的、合理的。以这种逻辑来论证某个结论便属于诉诸传统谬误。

如下面的几个例子：

“自古以来，在西方社会都是君权神授。所以，现在我们也不应该主张主权在民”

“中国社会一直都是女主内男主外，因此，妇女不该外出工作”

“我们传统的信仰是佛教，而佛教向来反对杀生，因此，我们因该废除死刑”

“俗话说：个人各扫门前雪，莫管他人瓦上霜。所以，王大爷在路边摔倒你不应该上去关心”。

“现在大家都在说邻居小罗勾结敌对势力，那他一定不是什么好人。所以，我们应该跟他划清界限，以后不相往来”。

以上例证都是以人数多少或时间跨度为的主要依据来支持其观点，其假设是：大多数人认为的、或传统的观点就是正确的。然而，一个观点被大多数人认同或持续了很长时间并不能充分证明其正确性。

历史上有很多观点都持续了很长时间，或被大部分人接受支持，但之后被证明是错误、甚至荒诞的，如：“地球是平的”，“太阳围绕地球转”，” 女人应该遵从三从四德 “，“君要臣死，臣不得不死”，“两个凡是”，“忠诚不绝对，就是绝对不忠诚”，“幸福的中国人民深情惦记着生活在水深火热之中的美国人民 ”，“人有多大胆，地有多高产”，“倾家荡产大搞钢铁”，“文化大革命好，就是好来就是好”，“一个萝卜千斤重，两头毛驴拉不动”，等等。

上述例证都犯了诉诸公众谬误，皆为谬论。

更多例证

  1. 小罗：为什么你认为男生优越于女生？

    小辑：因为我从小到大接受的都是这种观念。

    从不同的角度来看，上例含有多个逻辑谬误：熏鲱谬误（小辑把论题转移道他为何有这种观念），一厢情愿谬误（因为这是我接受的观念，所以这个观念是对的），以及诉诸公众（传统）谬误。

    小辑的隐含预设是：如果身边的人（或家庭成员）认同且灌输给我的观念，这种观念是正确的。显然这个假设并不成立，小辑也并没有给出其它合理可靠的理由来证明他的观念，所以他明显犯了诉诸公众谬误。

  2. 费人有与曾子同名族者而杀人。人告曾子母曰：‘曾参杀人。’曾子之母曰：‘吾子不杀人。’织自若。有顷焉，人又曰：‘曾参杀人。’ 其母尚织自若也。顷之，一人又告之曰：‘曾参杀人。’ 其母惧，投杼逾墙而走。夫以曾参之贤与母之信也，而三人疑之，则慈母不能信也。”

    –《战国策・秦策二》

    贤如曾参， 明如曾母，也不能幸免于此类谬误，以致诗人李白也曾叹: “曾参岂是杀人者？ 谗言三及慈母惊。”

    类似的感叹不胜枚举，如 “三人成虎事多有，众口铄金君自宽” 1，“且类曾参杀人，三人成虎，忧谗畏讥，不可终日” 2, “臣闻之，积羽沉舟，羣轻折轴（群轻折轴），众口铄金，积毁销骨” 3，等等。可见这种谬误盛行古今。理性之路，任重而道远。
